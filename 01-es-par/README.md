# Ejercicio 01 – EsPar (C#)

## Consigna
Crear un método que:
- Reciba un número entero
- Devuelva `true` si es par
- Devuelva `false` si es impar

Luego, desde el `Main`, recorrer los números del 1 al 10 usando `while`
y mostrar el resultado por consola.

## Conceptos aplicados
- Métodos con retorno (`bool`)
- Operador módulo (`%`)
- Bucle `while`
- Separación de lógica y ejecución
